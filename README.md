App, File & Folder Launch Hub
• Four access points — keyboard shortcuts, right-click menu, system menu bar and Dock/main app — to quickly launch apps, files, folders and custom functions. Open items with a single click without switching windows, and access over 100 built-in practical tools including app uninstaller and disk cleaner.
• Fix the pain point of hard-to-locate apps after macOS 26 removed the native quick app preview panel, while resolving inconvenient file/folder access and insufficient built-in features.
• Launch files, folders, apps and over 100 custom functions via keyboard shortcuts.
• Manage files, folders, apps and function menus from the system menu bar; one-click launch and custom grouping supported.
• Launch Hub auto-syncs names and icons of native apps, files and folders with auto categorization. Switch between flat and directory management modes.
• Dock layout displays files and folders side-by-side with apps.
• One-click right-click launch for apps, files and various system tools. Smart data sync between right-click menu and main app.

Built-in Toolkit (Constantly Updated):
- Add any app, file or folder via right-click or the app and launch instantly.
- Bulk uninstall apps, scan ten residual folders to fully delete leftover files.
- One-click deep storage cleanup: erase user/app/sandbox cache, logs, crash reports, Xcode build caches, temp files, quarantine records and hidden .DS_Store files.
- One-click full memory release.
- Adjust mouse tracking speed.
- Show/hide hidden files, create hidden folders.
- Timed shutdown, screen lock, log out, restart and power off.
- Toggle auto-hide for Dock and menu bar with one click.
- Create new files.
- Erase files directly without moving to Trash.
- Cut and paste files.
- Bulk copy filenames and full paths.
- Screenshot, Activity Monitor, Digital Color Meter.
- Open the directory of selected items in Terminal.
- Open multiple selected files at once.
- Jump to the path stored in clipboard.
- Batch calculate various file hash values.
- Count characters and file sizes stored in clipboard.
- Convert Simplified/Traditional Chinese, switch uppercase/lowercase.
- JSON formatting and compression.
- Batch convert images to JPG to save disk space.
- Quick access to Privacy & Security, Dock and menu bar auto-hide settings.

• Support automatic launch on Mac startup.
• Multiple sorting layouts to locate and launch programs instantly: smart recommendation, custom sorting, favorite pinning.
• Search memory recommendation.
• Privacy & Security: Only access authorized folders, all data stored locally.
• Convenient calendar view for dates, legal holidays, lunar calendar and 24 solar terms with weather info.
• Upload custom images as app wallpaper for personalized interface.
