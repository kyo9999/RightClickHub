RightClickHub - File Center

App, File & Folder Launch Center
● Four-way access via Right-Click Menu, System Menu Bar, Dock, and Main App to quickly launch apps, files, and folders. Open files with a single click without switching interfaces. Access over 100 built-in utilities like app uninstallers and disk cleaners.
● Solves users’ trouble of quickly finding apps after macOS 26 discontinued the native quick app access feature. It also streamlines file and folder access and enriches various practical shortcut functions.
● System Menu Bar manages opened files, folders, apps, and function menus with one-click access and customizable grouping.
● Launch Center auto-syncs native app, file, and folder names/icons with auto-categorization. Switch between flat or directory management views.
● Dock layout supports placing files and folders alongside apps.
● One-click right-click to launch apps, files, and system tools. Right-click menu and main app data sync intelligently.
Built-in Utility Toolkit (Continuously Updated):
  - Add any app, file, or folder via right-click or app for instant launch.
  - Batch uninstall apps, scan 10 major residual folders, and thoroughly remove leftover files.
  - One-click deep storage cleanup, clear user/app/sandbox caches, logs, crash reports, Xcode build caches, temp files, download quarantine records, and hidden .DS_Store files.
  - One-click full memory release.
  - Adjust mouse tracking speed.
  - Show/hide hidden files and create hidden folders.
  - Scheduled shutdown, lock screen, log out, restart, and power off.
  - One-click toggle for Dock and Menu Bar auto-hide.
  - Create new files.
  - Permanently erase files without moving to Trash.
  - Cut and paste files.
  - Batch copy file names and full paths.
  - Screenshot, Activity Monitor, Digital Color Meter.
  - Open Terminal to selected item's directory.
  - Open multiple selected files at once.
  - Jump to path stored in clipboard.
  - Batch calculate multiple file hash values.
  - Count characters and file sizes in clipboard.
  - Convert Simplified/Traditional Chinese and toggle uppercase/lowercase.
  - JSON formatting and minification.
  - Batch convert images to JPG to save disk space.
  - Quick access to Privacy & Security, Dock, and Menu Bar auto-hide settings.
● Supports auto-start on Mac boot.
● Multiple sorting layouts for instant app location: Smart Recommendations, Custom Sort, Favorites pinned to top.
● Search memory recommendations.
● Privacy & Security: Only accesses authorized folders with local storage.
● View dates, public holidays, Lunar calendar, and solar terms, plus calendar and weather info.
● Upload custom images as app wallpaper for a personalized interface.
